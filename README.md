# 🎯 FocusBento — Интеллектуальный Канбан-Органайзер

FocusBento — это современное веб-приложение для управления задачами в стиле Bento Grid, объединяющее методологию Канбан, элементы геймификации и искусственный интеллект для борьбы с прокрастинацией.

## 🚀 Основные фичи

- **Bento Dashboard**: Стильный модульный интерфейс с адаптивной сеткой.
- **Интерактивный Канбан**: Полноценный Drag-and-Drop (HTML5 API) для распределения задач по колонкам (План ➔ В работе ➔ Готово).
- **FocusAI Коуч**: Интеграция с OpenAI (модель `gpt-4o-mini`) через прокси-шлюз. Анализирует контекст доски и выдаёт ежедневные продуктивные советы.
- **ИИ-Декомпозиция**: Автоматическое дробление крупных абстрактных задач на конкретные подзадачи с расчётом времени в формате JSON.
- **Помодоро-таймер**: Интерактивный виджет фокусировки, который синхронизирует время тиканья прямо при клике на карточку задачи.
- **RPG-Геймификация**: Система прокачки профиля. Начисление XP за выполнение задач и сессии Помодоро с честным математическим пересчётом уровней (включая откат при отмене задач).
- **Умное автоудаление**: Автоматическая очистка выполненных задач из колонки «Готово» через 5 минут для предотвращения захламления доски.
- **Безопасная архитектура**: Полная защита приватных API-ключей через переменные окружения Vite (`.env.local`) без хардкода.

## 🛠️ Технологический стек

- **Frontend**: React, TypeScript, Tailwind CSS (v4)
- **Сборщик**: Vite
- **Backend / Database**: Supabase (PostgreSQL, Row Level Security, RPC)
- **ИИ-Провайдер**: OpenAI API (`gpt-4o-mini`) через ProxyAPI

## 📦 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com
cd focus-bento
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Настройка переменных окружения

Создайте в корне проекта файл `.env.local` и добавьте ваши ключи доступа:

```env
# Существующие ключи Supabase
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

# Ключ для работы ИИ-модуля (ProxyAPI)
VITE_AI_API_KEY=sk-dwGpjAqbxxSRkTZV0HTgD1Ux9ZkZSWXYr
```

### 4. Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5173`

## ⚙️ Особенности локальной разработки в РФ

Для стабильной работы синхронизации с Supabase требуется включённый VPN-туннель. Запросы к ИИ-шлюзу ProxyAPI настроены в обход VPN для предотвращения блокировок Cloudflare со стороны реселлера.
